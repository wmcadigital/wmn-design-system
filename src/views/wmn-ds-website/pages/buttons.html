Buttons

Buttons

Buttons and links are items used to perform an action on a page

It's important to know when to use a button or went to use a link

Buttons should be used in situations where a user might need to preform an action, such as:

  • Submitting a form
  • Uploading
  • Specify a next or previous step in a process
  • Begin a new task
